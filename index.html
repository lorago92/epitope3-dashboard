Simulation Methodology
ML regression signals computed using only data available at each rebalance date. LightGBM trained with 21-day purge gap — zero information leakage. 84 bi-weekly rebalances across 427 biotech/biopharma securities. Macro regime classification via PCA decomposition of a multi-asset panel into latent risk, rates, and inflation factors, classified by Hidden Markov Model. A secondary biotech factor (PC6) refines the 4-level gate: Bull+PC6>0 = 100%, Bull+PC6≤0 or Neutral+PC6>0 = 50%, Neutral+PC6≤0 = 25%, Bear = 0%. Bi-weekly rebalancing with equal-weight (1/N) top-20 positions and -30% stop loss per position. Selected through rigorous walk-forward validation (NB_R22→R29).
Live Paper Trading
Paper trading commenced March 2026 via Alpaca. The live pipeline runs on an automated schedule using identical logic to the simulation. The 8-week validation period ends April 27, 2026. Loading live portfolio status… All orders and P&L are logged and auditable.
